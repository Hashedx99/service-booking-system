# Service Booking System

## Project Description
The Service Booking System is a web application designed to facilitate the booking and management of property and service appointments. It supports multiple user types, secure authentication, and scheduling features for both property and service providers.

## Tools and Technologies
* Java 17
* Spring Boot
* Spring Data JPA
* Spring Security (JWT)
* Maven
* Thymeleaf (for email templates)
* Postman (for API testing)
* Git & GitHub
